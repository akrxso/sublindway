## 작품 설명
 본 작품은 지하철 이용 시 시각장애인이 겪는 어려움을 해결하고자 개발된 앱이다. 이 앱은 OCR 기술로 지하철 상하행선 역 정보를, YOLO 기술로 전방 객체를 인식해 시각장애인에게 안내한다. 또한 실시간으로 지하철 역 위치를 안내해 보다 더 나은 하차를 도운다. 
 안내는 진동 모터를 사용한 점자안내와 음성 안내로 정보 접근성을 높였으며 앱 제어는 사용자가 시각적 정보를 볼 수 없음을 고려해 음성 명령으로 앱을 제어할 수 있도록 했다. 
 보호자는 웹과 모바일 화면으로 사용자의 위치를 실시간으로 확인할 수 있게 하여 시각장애인의 지하철 이용이 보다 편리해지고 안전해지게 제작했다.

## 기대 효과
- 시각장애인의 독립적 지하철 이용으로 일상의 편의성 증진 기대
- 보호자가 웹 또는 모바일로 사용자의 실시간 위치를 확인 가능해 긴급상황 대응및 안전 확보
- 시끄러운 지하철 상황에 시각장애인 이외의 다양한 사용자들의 상용화 기대
