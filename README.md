<h1>GreenBoogie</h1>



## 기대 효과
- 시각장애인의 독립적 지하철 이용으로 일상의 편의성 증진 기대
- 보호자가 웹 또는 모바일로 사용자의 실시간 위치를 확인 가능해 긴급상황 대응및 안전 확보
- 시끄러운 지하철 상황에 시각장애인 이외의 다양한 사용자들의 상용화 기대
